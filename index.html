Mercyhurst Football — Defensive Stats (2025)

Overview

This project shows defensive stats for the Mercyhurst Football team in the year 2025. It compares each defensive player's year-end stats and their average tackles for the year. It also compares the team's stats for the selected week, and the opponent's team offensive stats for that week.

Web Scraping

Data was scraped off of the Mercyhurst athletcis webiste, hurstatheltics.com. The specific page use was https://hurstathletics.com/sports/football/stats. This page contained all the tables I scraped to get my data. Tools used in Python included BeautifulSoup, Pandas and Requests.

Database

The database created is consisted of three csv files. One containing Mercyhurst's defensive player's stats, one containing the opposing team's offensive stats,and one containing Mercyhurst's team defensive stats. From that database, two other csv files were created. One containing defensive player's average tackles per game and one containing the opposing teams rushing and passing attempts compared to Mercyhurst's total team tackles.
